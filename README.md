 ##City Sky Watch- The  Weather App

A full-stack weather application built with React.js for the frontend and Node.js for the backend. The app allows users to search for weather conditions in cities worldwide, view detailed weather information, and save their favorite cities for quick access.

## Features

- Search for weather by city or region
- Display current temperature, humidity, wind speed, and other weather metrics
- 5-day weather forecast
- Save favorite cities for quick access
- Responsive design that works on both desktop and mobile devices

## Technologies Used

### Frontend
- React.js
- CSS3 with Flexbox and Grid
- Local Storage for saving favorites
- React-Toastify for notifications

### Backend
- Node.js
- Express.js
- OpenWeatherMap API for weather data
